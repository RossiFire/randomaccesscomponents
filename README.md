## Overview

Random Access Components is a curated collection of React components, hooks, and utilities designed for Next.js and React applications. Every piece is crafted to be animated, accessible, and production-ready. Browse the docs, pick what you need, and drop it into your project.


## Documentation

Visit the [documentation](https://www.randomui.dev/docs/getting-started) website to browse components, view live demos, and copy the source directly into your project.
